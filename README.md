# RegaloShop

RegaloShop es una aplicación web tipo e-commerce orientada a la venta de regalos. 
Este proyecto representa una versión inicial enfocada en la estructura básica de una tienda online.

## 🎯 Objetivo
Desarrollar una plataforma que permita visualizar productos, agregarlos a un carrito de compras y simular una experiencia de compra básica.

## 🚀 Funcionalidades principales
- Visualización de catálogo de productos
- Detalle de productos
- Carrito de compras (simulado)
- Interfaz amigable para el usuario
